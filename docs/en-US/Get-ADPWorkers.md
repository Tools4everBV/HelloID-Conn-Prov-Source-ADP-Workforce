# Get-ADPWorkers

## SYNOPSIS
Retrieves the Workers and WorkerAssignments from ADP Workforce

## SYNTAX

```
Get-ADPWorkers [-ConfigurationSettings] <PSObject> [<CommonParameters>]
```

## DESCRIPTION
Retrieves the Workers and WorkerAssignments from ADP Workforce
